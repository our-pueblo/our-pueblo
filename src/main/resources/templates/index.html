What is Our Pueblo?

Our Pueblo is an application that was created as a way for volunteers to contribute to the translation of the entire City of San Antonio website.

How Does it Work?
1. Request

An admin requests a specific page to be translated.

2. Translate

Volunteers submit edits and corrections to the Google translation.

3. Review

Admins review user submissions to make sure they are ready for deployment.

It is just that simple! Try it for yourself!
